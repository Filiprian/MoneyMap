# MoneyMap

**MoneyMap** is a full‑stack web application that helps anyone gain crystal‑clear insight into their personal finances. By aggregating transactions, allowing budget creation, and visualizing data with interactive charts, MoneyMap turns raw numbers into an intuitive financial story—making users financially literate and in control.

---

## Demo

A live demo is included here: https://moneymap-lemon.vercel.app/

---

## Why MoneyMap?

- **Financial awareness** – Users instantly see where money comes from and where it goes.  
- **Clear visual storytelling** – Interactive graphs make trends obvious at a glance.  
- **Budget empowerment** – Set limits, track progress, and stay on top of spending.  
- **Language support** – Interface available in English and Czech, widening accessibility.  

---

## Features

| Category      | Description |
|---------------|-------------|
| **Dashboard** | Real‑time overview of total balance, income, and expenses. |
| **Budgets**   | Create, edit, and monitor budgets per category; visual progress bars show remaining funds. |
| **Transactions** | Add, edit, and delete individual transactions; filter by date, category, or amount. |
| **Interactive Charts** | Line, bar, and pie charts (Chart.js) that update instantly as data changes. |
| **Localization** | UI strings toggled between English and Czech. |
| **Demo Mode** | Uses Dexie (IndexedDB) for client‑side persistence, so reviewers can test the app without a backend. |
| **Responsive Design** | Works on desktop and mobile browsers. |
